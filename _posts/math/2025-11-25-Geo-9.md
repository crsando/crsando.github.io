### 1

*LS-7年级压轴*

如图，已知$\triangle ABC$与$\triangle DEF$为等腰三角形，$AB = AC$, $DE = DF$, $\angle BAF = \angle EDF$。

若$AD = AB$，求证：$AF = AE + BC$

![](https://crsando.github.io/images/2025-11-20/A-001.png)

***Solution***

![](https://crsando.github.io/images/2025-11-20/A-001-Ans.png)

由于$\angle BAC = \angle EDF$，所以$A,D,F,E$四点共圆，所以$\angle AED = \angle AFD$，这里想到利用$DE = DF$，可以把$\triangle AED$旋转过去，将$DE$转到$DF$，$A \rightarrow P$，那么很自然得到$DA = DP$，利用旋转算角度可知$\angle ADP = \angle BAC$，所有$\triangle BAC \cong \triangle ADP$

*2025 浙江名校联考填空压轴*

如图，$AB = AC$, $\angle ADC = 60^\circ$, $AE = BE$, $\angle BAC = 96^\circ$, 求$\angle DCE$

![](https://crsando.github.io/images/2025-11-20/A-002.png)

***Solution***

![](https://crsando.github.io/images/2025-11-20/A-002-Ans.png)

角多算角，先算一下发现$\angle ACD = 42^\circ$，$\angle DAC = 78^\circ$, $\angle EAB = \angle EBA = 18^\circ$。

这里发现$78^\circ - 18^\circ = 60^\circ$，所以很自然想到左右对称。

补齐一个正三角$\triangle ADP$，在$AP$上取$AF = AE$，那么$\triangle AEB \cong \triangle AFC$，所以$AF = EF = FC$，所以$F$是三角形$\triangle ACE$的外心。这里计算一下可知$\angle EFC = 2\angle EAC = 2 \times 78^\circ = 156^\circ$，所以$\angle FEC = 12^\circ$。这里显然平行，所以$\angle FEC = \angle ECD =12^\circ$。

[Source](https://www.bilibili.com/video/BV1Km14BNEp8)

*2025 人大附中初二期中压轴*

如图，在 $\triangle ABC$ 中，$AB = AC$，点 $D$ 在边 $AC$ 上，$\angle DBC = 45^\circ$，点 $E$ 在边 $BC$ 延长线上，满足 $\angle BDC = \angle BAE$。

1. 判断$BD$和$AE$的位置关系并证明。
2. 过$E$作$EF \perp AE$，交$AC$的延长线于$F$，请补全图形，并说明$AB, AD, AF$的之间的数量关系。
3. 若$BE = 2$，点$P,Q$分别是$BE, AE$上的动点，求$DP + PQ$的最小值。

![](https://crsando.github.io/images/2025-11-20/A-003.png)

***Solution***

第一问，利用公共角算角可知$\angle AEB = \angle DBE = 45^\circ$，所以$\triangle BHE$是等腰直角三角形，所以$BD \perp CE$

第二问，算角显然可知$EF \parallel BD$，作$DG \perp EF$于$G$，可知$DG = EH$，结合等角，可知$\triangle DGF \cong \triangle BGA$，所以$DF = AC$，所以$AD + AB = AF$

第三问，注意到$EA$和$EF$这两条直线关于$EB$对称，所以我们找$Q$在$EF$上的对称点$Q'$，则$DP + PQ = DP + PQ' \geqslant DQ' \geqslant DH = EH = \sqrt{2}$，即为所求最小值。

### 4

[Source](https://www.bilibili.com/video/BV1wxMEzoEPt)

如图，正方形 $ABCD$ 边长为 $8$，$E、F$ 分别为 $AB、CD$ 边动点，且 $AE = CF$，$BG \perp EF$ 于点 $G$，则 $AG$ 的最小值为 $\underline{\ \ \ \ \ }$。

![](https://crsando.github.io/images/2025-11-20/A-004.png)

***Solution***

![](https://crsando.github.io/images/2025-11-20/A-004-Ans.png)

条件里的$AE = CF$实际上是说$EF$过$ABCD$的几何中心$M$，那么$G$点的轨迹就是$BM$为圆心的圆，记圆心为$O$，则$AG \geqslant AO - OG$

这里分别计算一下，$BD = 8\sqrt{2}$, $BO = 2\sqrt{2}$，

$AO$可以用余弦定理计算，也可以勾股

$$
    AO^2 = AM^2 + OM^2 = OM \sqrt{5} = 2\sqrt{10}
$$

所以$AG \geqslant 2(\sqrt{10} - \sqrt{2})$
