# 💊 关于补剂的真相：我的自用方案与避坑指南

> **前言**
>
> 圈里不少人在吃补剂，但中文互联网上关于补剂的内容大部分是收了钱的推广，有用的信息被埋在一堆软文里。这篇文章是我自己查资料整理的，写出来也给大家做个参考。
>
> 去年做完手术之后，我把市面上能找到的补剂基本查了一遍，翻了上百篇临床研究，最终筛出一套自己在用的方案，顺便也列了一份不值得买的清单。

**立场声明：** 补剂不是药物，也不是安慰剂。它的定位是**弥补生活方式造成的营养缺口**。

长期熬夜、饮食不规律、持续高压的人，合理使用补剂可以在一定程度上改善生理状态，但前提是：你了解**作用机制**、认识**证据强度**，知道自己在吃什么。

---

## 第一部分：6 种有临床证据支持的补剂

**背景：** 加密货币从业者和金融交易员的生活方式有几个共性问题：
*   **昼夜节律紊乱：** 导致内源性褪黑素分泌异常和维生素 D 合成不足；
*   **心理应激状态：** HPA 轴持续激活，皮质醇水平偏高，系统性炎症标志物上升；
*   **饮食结构问题：** 单一或不规律，微量营养素摄入不足。

以下 6 种补剂按证据等级排序，均为我目前每天在用的。

### 1. 鱼油（Omega-3 多不饱和脂肪酸）
*   **证据等级：** 🟢 **A**
*   **建议剂量：** 每日推荐摄入 **2-3g EPA+DHA 总量**。

**核心机制：**
1.  **抗炎：** EPA 和 DHA 是消退素和保护素的前体，能下调 IL-6、TNF-α 等促炎细胞因子。*Bernasconi 等人 (2021)* 的 Meta 分析显示，每日补充 Omega-3 可显著降低心血管死亡率和心肌梗死风险。
2.  **神经保护：** DHA 占大脑皮层磷脂中脂肪酸总量的约 25%。多项研究显示 EPA/DHA 对工作记忆和执行功能有改善作用，尤其在慢性压力人群中效果明显。

> **🛒 选购要点**
> *   优先选 **甘油三酯型（TG form）**，生物利用度优于乙酯型（EE form）。
> *   认准 **EPA+DHA 含量标注**（注意不是胶囊总重）。
> *   **冷藏保存**，防止氧化产生有害的脂质过氧化物。

### 2. 维生素 D3 + K2
*   **证据等级：** 🟢 **A**
*   **建议剂量：** D3 每日 2000-4000 IU，K2 每日 100-200 mcg（MK-7 型）。
*   **目标值：** 血清 25(OH)D 维持在 40-60 ng/mL。

**核心机制：**
*   **D3 (免疫与情绪)：** 维生素 D 受体广泛分布于免疫细胞。*Anglin 等人 (2013)* 研究显示缺乏 D 与抑郁风险相关；*Zhu 等人 (2020)* 确认补充 D 可改善焦虑。
*   **K2 (钙引导)：** D3 促进钙吸收，K2（尤其是 MK-7）激活骨钙素和基质 Gla 蛋白，将钙引导至骨骼并抑制血管钙化。**单补 D3 不补 K2 存在血管钙化风险。**

> **🛒 选购要点**
> *   选 **K2-MK7 亚型**（半衰期长，血药浓度稳定）。
> *   均为脂溶性，**随含脂肪的餐食服用**。

### 3. 姜黄素（Curcumin）
*   **证据等级：** 🟡 **B+**
*   **建议剂量：** 每日 500-1000 mg。
*   **关键条件：** 必须含 **胡椒碱（piperine）** 或采用 **脂质体/纳米粒包裹技术**。

**核心机制：**
*   **抗炎：** 抑制 NF-κB 信号通路及 COX-2 活性。
*   **生物利用度：** 游离姜黄素吸收率极低（<1%），胡椒碱可将其提高约 2000%。
*   **脑部健康：** *Small 等人 (2018)* 研究显示，长期服用生物可利用姜黄素可改善记忆评分，减少脑部淀粉样蛋白沉积。

> **🛒 选购要点**
> *   普通姜黄粉基本无效。
> *   **随含脂肪的餐食服用**。

### 4. 辅酶 Q10（CoQ10）
*   **证据等级：** 🟡 **B+**
*   **建议剂量：** 每日 100-200 mg。

**核心机制：**
*   **能量代谢：** 参与线粒体 ATP 生成。
*   **心脏保护：** *Mortensen 等人 (2014)* 的 Q-SYMBIO 试验显示，补充 CoQ10 显著降低慢性心衰患者的不良心血管事件。
*   **他汀伴侣：** 服用他汀类药物会抑制体内 CoQ10 合成，补充它是必要的以减少肌痛副作用。

> **🛒 选购要点**
> *   选 **还原型泛醇（Ubiquinol）**，吸收率高于氧化型泛醌（Ubiquinone）。
> *   **随餐服用**。

### 5. 甘氨酸镁（Magnesium Glycinate）
*   **证据等级：** 🟢 **A**
*   **建议剂量：** 每日 200-400 mg **元素镁**（甘氨酸镁中元素镁约占 14%）。

**核心机制：**
*   **神经镇静：** 镁是 NMDA 受体的阻断剂，并调节 GABA-A 受体，具有镇静和抗焦虑效果。
*   **助眠：** *Abbasi 等人 (2012)* 证实补充镁可改善失眠患者的睡眠效率。
*   **为何选甘氨酸镁：** 甘氨酸本身是抑制性神经递质；肠道耐受性好，不易腹泻。

> **🛒 选购要点**
> *   **睡前 1-2 小时服用**最佳。
> *   若腹泻，请减量。

### 6. 复合维生素（Multivitamin）
*   **证据等级：** 🟡 **B**
*   **定位：** 营养保险，兜底手段。

**核心机制：**
*   *Vyas 等人 (2024)* 的 COSMOS 试验显示对老年人认知功能有小幅改善。
*   对于饮食不规律（常吃外卖）的人群，是防止微量营养素缺乏的最简单手段。

> **🛒 选购要点**
> *   选 **甲基化 B 族**：B12 选甲基钴胺素，叶酸选 5-甲基四氢叶酸（5-MTHF）。
> *   **避免超大剂量（Mega-dose）**，防止脂溶性维生素蓄积中毒。

## 第二部分：7 种不推荐购买的补剂（避坑清单）

以下产品并非完全无活性，而是对**健康人群**来说，现有证据不支持其声称功效，性价比极低。

| 补剂名称 | 不推荐理由 | 备注 |
| :--- | :--- | :--- |
| **1. 口服胶原蛋白** | **无高质量证据支持对皮肤有益。** 口服后被水解为氨基酸，无组织特异性，和吃鸡胸肉没本质区别。 | 补充维C、锌铜及防晒更有效。 |
| **2. 苹果醋** | **减重效果微弱且研究质量低。** 长期饮用高酸性液体损害牙釉质和食道。 | 控制碳水和运动更有效。 |
| **3. 排毒/肝脏清洁产品** | **“排毒”是伪科学。** 肝脏不需要“清洗”。部分草本成分反而可能导致药物性肝损伤（DILI）。 | 护肝核心是戒酒和少吃药。 |
| **4. 藤黄果** | **减重效果极有限。** 与安慰剂无显著差异。多数阳性研究存在发表偏倚。 | 有肝损伤病例报告。 |
| **5. 绿茶提取物 (减肥用)** | **临床意义极小。** 热量消耗增加不到 100 千卡/天。 | 高剂量 EGCG 有肝损伤风险。喝茶即可。 |
| **6. 奶蓟草/水飞蓟** | **对健康人群无预防保护作用。** “解酒”缺乏证据，酒精代谢由基因决定。 | 仅对已有肝病患者可能有辅助价值。 |
| **7. 左旋肉碱 (燃脂用)** | **不运动吃了没用。** 健康人可内源合成。仅配合运动时有微弱效果。 | 限速步骤不在肉碱而在能量消耗。 |

## 第三部分：补剂使用的基本原则

1.  **先处理生活方式：** 睡眠、饮食、运动不及格，补剂没用。
2.  **优先补充高风险缺口：** 维 D、镁、Omega-3 是现代人普遍缺乏的“三巨头”。
3.  **留出观察窗口：** 至少持续使用 4-8 周再评价效果。
4.  **用数据说话：** 每年验血（25(OH)D、血清镁、血脂谱），根据数据调整。
5.  **控制剂量上限：** 警惕脂溶性维生素（A、D、E、K）蓄积中毒。

### 📅 我的每日服用安排

*   **☀️ 早餐后：** 复合维生素 + 维生素 D3/K2 + 鱼油 + 辅酶 Q10
    *   *理由：均为脂溶性，随餐吸收好。*
*   **🕛 午餐后：** 姜黄素
    *   *理由：需要脂肪辅助吸收。*
*   **🌙 睡前 1 小时：** 甘氨酸镁
    *   *理由：助眠，且分段服用减轻肠胃负担。*
